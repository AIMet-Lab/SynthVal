synthval.features_extraction.MambaFeatureExtractor#

class synthval.features_extraction.MambaFeatureExtractor(model_id)#

Bases: FeatureExtractor

Feature extractor using models from the HuggingFace MambaVision family (https://huggingface.co/collections/nvidia/mambavision-66943871a6b36c9e78b327d3).

Note: MambaVision models require a CUDA-enabled environment and the installation of specific packages. To use these models as feature extractors, users must have CUDA installed and install the necessary packages by running pip install mambavision.

Parameters:

model_id (str) –

model_id#

HuggingFace model ID for the selected MambaVision model.

Type:

str

feature_extraction(image)#

Extract features from a PIL image using the selected HuggingFace MambaVision model.

Parameters:

image (PIL.Image.Image) – The image to extract features from.

Returns:

A 1-D NumPy array of 640 features.

Return type:

numpy.ndarray
